# Sistema de Matrícula Académica en Java

## Descripción del proyecto
El Sistema de Matrícula Académica es una aplicación desarrollada en Java con fines educativos, cuyo objetivo es automatizar el registro y la gestión básica de la información de los estudiantes durante el proceso de matrícula académica. El sistema funciona a través de un entorno de consola y permite una interacción sencilla con el usuario.

## Objetivo
Desarrollar un sistema en Java que permita registrar y gestionar la matrícula académica de estudiantes, aplicando los conocimientos fundamentales de programación adquiridos durante el curso.

## Funcionalidades
- Registro de datos básicos de los estudiantes.
- Gestión básica del proceso de matrícula académica.
- Visualización de la información ingresada.
- Menú interactivo por consola.
- Validación básica de datos.

## Tecnologías utilizadas
- Lenguaje de programación: Java
- Entorno de desarrollo: NetBeans / IntelliJ IDEA / Visual Studio Code
- Control de versiones: Git
- Repositorio remoto: GitHub

## Requisitos para la ejecución
- Tener instalado Java Development Kit (JDK).
- Contar con un entorno de desarrollo o ejecutar desde consola.

## Instrucciones de uso
1. Descargar o clonar el repositorio.
2. Abrir el proyecto en un entorno de desarrollo compatible con Java.
3. Ejecutar la clase principal del sistema.
4. Seguir las instrucciones que aparecen en la consola.
